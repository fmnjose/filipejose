It_Spreads

This game is the result of a personal challenge to learn the Godot game engine. The starting point was a randomly generated word - "spreads" - and I proposed to myself to finish the game in 1 week. A kind of personal game jam.

This was a really fun project that challenged me to conceptualise a game, design and implement different game systems, and also fuel my creativity through level design.

Be sure to try the game on a desktop as it was not designed with mobile devices in mind.
