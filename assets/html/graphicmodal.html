Graphic Playground

I've always been interested in using technology in creative ways. While learning about shaders, I was introduced to the concept of generative art and how it is possible to use mathematics and computer graphics to create unique images.

This led me to learn more about shaders and graphics development, and to work on a small project using C++ and OpenGL that consists of a simple playground (which I intend to expand) to experience this type of art that I find amazing.

I also took the opportunity to learn about compiling applications to WebAssembly, so I used Emscripten to build a web version.

Be sure to try the project on a desktop, as it was not designed to run on mobile devices.
