System Engineer at Celfocus

My first work experience was as a Systems Engineer at Celfocus, a Portuguese consulting company, working on projects with various telecommunications clients.

First, I was part of the release management team for a client's web portal, where I was responsible for managing development and pre-production environments, deployment pipelines, and articulating the release process with developers and QA teams.

I then joined a digital transformation project where I was responsible for provisioning Kafka and Cloudbees clusters for multiple development, pre-production and production environments. There was also a strong focus on building robust documentation of the systems and processes.

The last project I worked on was designing and implementing automated backup systems using Python scripts, ansible playbooks and interacting with Amazon S3 for data storage.

Working at Celfocus was a great experience for developing core skills like communication, inter-team operability and expectation management, while being pushed to learn and explore a lot of new technologies, which is the type of environment I really thrive in.
