Task & Maintenance Manager

This is a freelance project I worked on for a self-service car wash. The client needed a system that would allow him to schedule and manage the tasks of the operators, and also track all the data regarding the maintenance of the machines.

I developed a web application using Golang for the backend server and Google's Firestore for data storage. The interface is rendered on the server using Templ (https://github.com/a-h/templ) for HTML templating, and also HTMX to facilitate interaction.

The application was hosted on GCP using App Engine.
